# План структуры лендинга УралБелЭнерго

**Дата:** 10.06.2026
**Основа:** Research.md (10 разобранных аналогов + критика и чек-лист)

## Стек
- HTML5 + Tailwind CSS (через CDN) + минимум vanilla JS — один файл `index.html`, без сборщиков и фреймворков
- Шрифты: Google Fonts — Montserrat (заголовки) + Inter (текст)
- Иконки: инлайн SVG (без внешних иконных CDN — надёжнее)
- Изображения продукции: реальных фото нет → используем схематичные SVG-иконки арматуры (задвижка, клапан, охладитель пара и т.п.) в едином линейном стиле

## Палитра
- Тёмно-синий (основной): `#14375E`
- Белый / светло-серый фон: `#FFFFFF` / `#F4F6F8`
- Текст: `#25303B`
- Серый (второстепенный): `#6B7280`, `#E2E8F0`
- Акцент CTA: оранжевый `#FF7A00`

3. **Преимущества** — 6 карточек с иконками: официальный дилер ЧЗЭМ; на рынке с 2000 года; поставки для энергетики, нефтегаза, химии, металлургии; сертифицированная продукция; индивидуальный подбор по параметрам; работа с крупными промышленными заказчиками.

5. **О заводе ЧЗЭМ / статус дилера** — «Мы — официальный коммерческий представитель АО «Энергомаш (Чехов) – ЧЗЭМ»», описание завода (крупнейший поставщик трубопроводной арматуры на «пар-вода» для энергетики, в т.ч. атомной, нефтегаза, химии) + стилизованный бейдж «Сертифицированный дилер» (без скана конкретного документа — см. примечание ниже).

8. **Опросный лист (подбор арматуры)** — интерактивная форма на странице (тип арматуры, Ду, Ру, температура, среда) с кнопкой «Сформировать запрос» — визуально, без реальной отправки.

9. **Клиенты** — блок «Нам доверяют»: текстовые карточки компаний (ПАО «Фортум», ПАО «Юнипро», АО «Интер РАО-Электрогенерация», АО «Уральская Сталь», ЕвразХолдинг).

10. **Этапы работы** — таймлайн: Заявка → Подбор и расчёт → КП → Договор и поставка → Сервисная поддержка.

11. **Форма заявки/КП** — основной CTA: имя, компания, телефон/email, описание задачи, поле для файла (UI без реальной загрузки), кнопка «Отправить заявку» — JS показывает сообщение «Спасибо, заявка принята» (визуально, без реальной отправки).
